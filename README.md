# 📚 Banco de Dados - Materiais de Estudo

Repositório criado para organizar os materiais da disciplina de **Banco de Dados**, com base no livro:

> **Elmasri, Ramez; Navathe, Shamkant B.**  
> *Fundamentals of Database Systems*. 7ª ed. Pearson, 2016.

## 📁 Estrutura do Repositório
 ``
 /
├── Hands On/ # Atividades práticas (Hands On)
│ ├── enunciado/ # Enunciados das atividades
│ └── resolucao/ # Minhas resoluções
│
├── AS/ # Atividades Avaliativas (AS)
│ ├── enunciado/ # Enunciados das AS
│ └── resolucao/ # Minhas resoluções
│
├── Provas/ # Imagens de provas antigas
│
├── Resumos/ # Resumos por capítulo do livro
│ ├── Capitulo_1/
│ ├── Capitulo_2/
│ └── ...
│
``

## 📖 Detalhamento das Pastas

### 🖐️ Hands On
Exercícios aplicados.  
- Cada pasta contém o enunciado e a resolução do Hands On em questão.

### 📝 AS (Atividades Avaliativas)
Atividades que valem nota na disciplina.  
- Organizadas da mesma forma que as Hands On.

### 📸 Provas
Imagens (fotos/scans) de provas anteriores para treino e revisão.

### 📑 Resumos
Resumos teóricos baseados no livro do Elmasri & Navathe.  
- Cada capítulo tem sua própria pasta.  
- Dentro de cada pasta, um **PDF** contendo:
  - Resumo do capítulo
  - Resolução das atividades/exercícios do capítulo

## 🔗 Referência Bibliográfica

- **ELMASRI, R.; NAVATHE, S. B.** *Fundamentals of Database Systems*. 7. ed. Boston: Pearson, 2016.
